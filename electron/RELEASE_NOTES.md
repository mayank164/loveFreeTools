# 公益平台 v1.0.0 Release Notes

## 发布日期
2026年1月1日

## 下载
- **安装版**: `公益平台-1.0.0-x64.exe` - 完整安装程序，支持自定义安装路径
- **便携版**: `公益平台-1.0.0-Portable.exe` - 免安装，双击即用

## 系统要求
- Windows 10/11 (64位)
- 网络连接（应用需要连接互联网）

---

## 功能介绍

### 临时邮箱服务
- 多域名支持，一键生成临时邮箱地址
- 实时接收邮件，自动刷新
- 邮件历史记录管理
- 24小时自动清理，保护隐私

### AI 智能分析（Cloudflare Workers AI）
- 自动提取验证码，一键复制
- 邮件摘要生成（中英双语）
- 垃圾邮件智能检测
- 多语言翻译（中、英、日、韩、法、德、西、俄）
- URL 安全性检测

### 短链接服务
- 长链接转短链，方便分享
- 支持自定义短码（3-20字符）
- 访问次数统计
- 可设置过期时间
- 安全风险提示

### GitHub 加速
- Git clone 加速下载
- 仓库文件代理访问
- Release 文件加速

### 文件加速下载
- 支持任意 HTTPS 文件加速
- 无文件大小限制
- 支持断点续传
- 保留原始文件名

### 发送邮件
- 支持通过平台发送邮件
- IP 地理位置记录

## 应用特性

### 桌面客户端优势
- 原生 Windows 应用体验
- 系统托盘常驻，快速访问
- 菜单栏快捷操作
- 单实例运行，防止重复打开

| Alt+F4 | 退出应用 |
